# 3 Stupid Men Landing Page

Nowoczesny landing page dla 3SM Studio zbudowany w oparciu o Next.js 16, React 19, Tailwind CSS 4 i next-intl.

## Tech stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS 4
- next-intl
- Zod
- Radix Slot
- Lucide React
- ESLint
- Biome

## Wymagania

- Node.js 24+
- pnpm 10+

## Package manager

Ten projekt używa **wyłącznie pnpm**.

Jeśli spróbujesz użyć `npm`, instalacja zostanie zablokowana.

## Start projektu

Zainstaluj zależności:

```bash
pnpm install
```
